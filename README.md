Este projeto implementa um sistema bancário simples em Python, que permite realizar operações básicas como depósito, saque e visualização do extrato.
O programa utiliza estruturas de controle como while, if, elif e for para simular o funcionamento de um banco de forma interativa no terminal.

#Funcionalidades principais:

Registrar depósitos e atualizar o saldo.

Permitir saques com verificação de saldo suficiente.

Exibir o extrato completo das movimentações realizadas.

Encerrar o programa de forma segura.
